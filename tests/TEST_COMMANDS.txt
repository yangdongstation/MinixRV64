========================================
MinixRV64 测试命令列表
========================================

启动后，在 minix# 提示符下依次尝试以下命令：

基本测试：
----------
help
echo Hello, MinixRV64!
uname
ps
pwd

功能测试：
----------
echo Testing UART driver
echo Test 1 2 3
clear
help

系统信息：
----------
uname
ps

文件系统命令（当前未挂载）：
----------------------------
ls
cat test.txt
mkdir /tmp/test
pwd

多参数测试：
------------
echo one two three four five
echo Testing with special chars: !@#$%

长命令测试：
------------
echo This is a very long command to test the input buffer and see how it handles longer text

退出QEMU：
----------
按 Ctrl+A 然后按 X

========================================
快速复制粘贴测试
========================================

以下是可以直接复制粘贴的测试序列（注意：QEMU可能不支持粘贴，需要手动输入）：

help
echo Test 1
uname
ps
